GraphQL
=======

  GraphQL is an open-source data query and manipulation language for APIs, 
  and a runtime for fulfilling queries with existing data. `[wikipedia]`_

The Kenshi Deep Index endpoint for GraphQL provides an accelerated and edge
optimized access point for querying and retrieving blockchain data using GraphQL.

To learn more about GraphQL itself, you can visit the `official GraphQL website`_.

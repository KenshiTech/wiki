MQL (MongoDB Query Language)
============================

The Kenshi Deep Index endpoint for MQL provides an accelerated and edge
optimized access point for querying and retrieving blockchain data using the
MongoDB Query Language (MQL).

To learn more about MQL itself, you can visit the `official MongoDB website`_.
